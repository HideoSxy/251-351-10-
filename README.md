\# Проект \*Название\*

Данное приложение позволяет авторизовываться на сервере для многих клиентов (пользователь, админ) и использовать функционал программы: шифрование rsa, криптографическую хэш-функцию SHA-384, метод хорд для нахождения корней уравнения.

Все данные функции буду работать в win app подключаясь через сервер к приложению и бд, используя docker. Код написан на C++, Qt Framework.
